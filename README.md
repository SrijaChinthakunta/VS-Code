## üéØ Getting Started

### Prerequisites

- Node.js (v14 or higher)
- Modern web browser with Web Speech API support (Chrome recommended)
- Microphone for voice input

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/yourusername/vocal-studio-code.git
   ```

2. Navigate to the project directory:
   ```bash
   cd vocal-studio-code
   ```

3. Install dependencies:
   ```bash
   npm install
   ```

4. Start the development server:
   ```bash
   npm start
   ```

5. Open your browser and navigate to `http://localhost:3000`

## üó£Ô∏è Voice Commands

Vocal Studio Code understands a wide range of programming commands:

### Basic Commands
- **"Create variable x equals 10"** ‚Üí `x = 10`
- **"Print hello world"** ‚Üí `print("hello world")`
- **"Define function calculate"** ‚Üí `def calculate():`

- **"Run a loop from 1 to 10 and print i"** ‚Üí `for i in range(1, 11): print(i)`
- **"Loop from 5 to 15"** ‚Üí `for i in range(5, 16): # loop body`

### Control Flow
- **"If x is greater than 5 then"** ‚Üí `if x > 5: # code here`
- **"Create if statement condition then"** ‚Üí `if condition: # code here`

### Data Structures
- **"Create dictionary user"** ‚Üí `user = {}`
- **"Create array items"** ‚Üí `items = []`

### Editor Commands
- **"Delete last 2 lines"** ‚Üí Removes last 2 lines of code
- **"Clear everything"** ‚Üí Clears all code from editor
- **"Run the code"** ‚Üí Executes current code in terminal
